RayaGate Task
This project is a React application that fetches posts from the JSONPlaceholder API

##Content

-Fetches and displays a list of posts from the JSONPlaceholder API.
-Allows users to view post details and comments in a modal.
-Includes a search bar to filter posts by title.
-Uses React Context API for state management.

##Steps to run the projects

1- Install Node.js and npm.
2- Clone the repository: git clone https://github.com/your-username/rayagate-task.git
3- cd rayagate-task
4- npm install
5- npm start
6- Open your web browser and navigate to http://localhost:3000.
